1. What is DOM ?
- Document Object Model
- Structured representation of HTML Document
- Used to connect the webpage to scripts like javascript

2. Events: 
- notification that are sent to notify the code that something
happens in the page
- Event only triggered when the execution stack is empty, if not
the execution stack has to be empty
- There is a message queue
- go to this website for all the events: 
https://developer.mozilla.org/vi/docs/Web/Events

3. DRY principle: 
- Don Repeat Yourself
- Means if you need to use a repetitive block of code, 
store it in a function and call it where you need to use it
